列表初级

定义

  • 列表是按顺序排列的元素组成。
  • 列表直接的元素可以没有任何关系。元素类型可以不同
  • 列表的元素也可以是个列表，即大列表里面有小列表。

示例

xxxxxxxxxx
bicycles = ['trek','cannondale','redline','specialized']
print(bicycles)

打印结果

xxxxxxxxxx
['trek', 'cannondale', 'redline', 'specialized']

列表元素操作

查询

  • 正向访问

列表是有序的，用元素位置访问即可。

注意列表里的元素位置是从0开始，0，1，2 ...

xxxxxxxxxx
bicycles = ['trek','cannondale','redline','specialized']
print(bicycles[0])
  • 倒序访问

-1代表最后一个元素，-2代表倒数第二个元素

xxxxxxxxxx
bicycles = ['trek','cannondale','redline','specialized']
print(bicycles[-1])

修改

将元素重新赋值

xxxxxxxxxx
motorcycles = ['honda','yamaha','suzuki']
print(motorcycles)
motorcycles[0] = 'ducati'
print(motorcycles)

结果：

xxxxxxxxxx
['honda', 'yamaha', 'suzuki']
['ducati', 'yamaha', 'suzuki']

增加

  • 在列表末尾添加元素

    xxxxxxxxxx
    motorcycles = []
    motorcycles.append('honda')
    motorcycles.append('yamaha')
    motorcycles.append('suzuki')
    print(motorcycles)
    xxxxxxxxxx
    ['honda', 'yamaha', 'suzuki']

  • 根据元素位置插入元素

    xxxxxxxxxx
    motorcycles = ['honda']
    motorcycles.insert(0,'ducati')
    print(motorcycles)
    xxxxxxxxxx
    ['ducati', 'honda']

删除

  • 根据元素位置删除元素

    xxxxxxxxxx
    motorcycles = ['honda','yamaha','suzuki']
    print(motorcycles)
    del motorcycles[1]
    print(motorcycles)
    xxxxxxxxxx
    ['honda', 'yamaha', 'suzuki']
    ['honda', 'suzuki']

  • 使用pop()删除元素，并返回元素

    • pop()删除末尾元素
    xxxxxxxxxx
    motorcycles = ['honda','yamaha','suzuki']
    print(motorcycles)
    popped_motorcycle = motorcycles.pop()
    print(motorcycles)
    print(popped_motorcycle)
    xxxxxxxxxx
    ['honda', 'yamaha', 'suzuki']
    ['honda', 'yamaha']
    suzuki
    • pop(index) 删除index位置上的元素
    xxxxxxxxxx
    motorcycles = ['honda','yamaha','suzuki']
    print(motorcycles)
    popped_motorcycle = motorcycles.pop(1)
    print(motorcycles)
    print(popped_motorcycle)
    xxxxxxxxxx
    ['honda', 'yamaha', 'suzuki']
    ['honda', 'suzuki']
    yamaha

  • 根据值删除元素(只会按顺序匹配第一个相同元素值删除)

    xxxxxxxxxx
    motorcycles = ['honda','yamaha','suzuki','ducati']
    print(motorcycles)
    motorcycles.remove('yamaha')
    print(motorcycles)
    xxxxxxxxxx
    ['honda', 'yamaha', 'suzuki', 'ducati']
    ['honda', 'suzuki', 'ducati']

组织列表

sort()进行永久性排序

  • 正序排序 （字符串按首字母，数字按大小)

    xxxxxxxxxx
    cars = ['bmw','audi','toyota','subaru']
    cars.sort()
    print(cars)
    xxxxxxxxxx
    ['audi', 'bmw', 'subaru', 'toyota']

  • 倒序排序

xxxxxxxxxx
cars = ['bmw','audi','toyota','subaru']
cars.sort(reverse=True)
print(cars)
xxxxxxxxxx
['toyota', 'subaru', 'bmw', 'audi']

sorted()进行临时性排序,即不改变原始列表顺序

xxxxxxxxxx
cars = ['bmw','audi','toyota','subaru']
print(cars)
print(sorted(cars))
print(cars)
xxxxxxxxxx
['bmw', 'audi', 'toyota', 'subaru']
['audi', 'bmw', 'subaru', 'toyota']
['bmw', 'audi', 'toyota', 'subaru']

倒着打印列表

  • 只是将列表倒着打印，并不是倒着排序
  • 列表永久性进行了改动
xxxxxxxxxx
cars = ['bmw','audi','toyota','subaru']
print(cars)
cars.reverse()
print(cars)
xxxxxxxxxx
['bmw', 'audi', 'toyota', 'subaru']
['subaru', 'toyota', 'audi', 'bmw']

列表解析

  • 列表解析一种python语法支持的列表操作方式
  • 优点是代码简洁，可将多行代码实现功能放在一行中
  • 面对初学者并非都会介绍列表解析，但是在阅读别人代码中很可能会碰到。

先通过for循环创建一个列表

xxxxxxxxxx
squares = []
for value in range(1,11):
   square = value**2
   squares.append(square)
print(squares)
xxxxxxxxxx
[1, 4, 9, 16, 25, 36, 49, 64, 81, 100]

列表解析写法

xxxxxxxxxx
squares = [value**2 for value in range(1,11)]
print(squares)

使用列表的一部分

切片

  • 列表[f_index:l_index]：从f_index到l_index(不包括l_index)取出列表的子集。
xxxxxxxxxx
players = ['charles','martina','michael','florence','eli']
print(players[1:3])
xxxxxxxxxx
['martina', 'michael']
  • 列表[:l_index]：从头到l_index(不包括l_index)开始取出列表的子集。
xxxxxxxxxx
players = ['charles','martina','michael','florence','eli']
print(players[:3])
xxxxxxxxxx
['charles', 'martina', 'michael']
  • 列表[f_index:]: 从f_index返回到末尾列表自己
xxxxxxxxxx
players = ['charles','martina','michael','florence','eli']
print(players[2:])
xxxxxxxxxx
['michael', 'florence', 'eli']

  • 其他

    • 输出列表最后三个元素
xxxxxxxxxx
players = ['charles','martina','michael','florence','eli']
print(players[-3:])
xxxxxxxxxx
['michael', 'florence', 'eli']

复制列表

列表[:]: 代表整个列表

xxxxxxxxxx

print(my_foods)
print(friend_foods)
​
xxxxxxxxxx

['pizza', 'falafel', 'carrot cake']

 

注意点

使用切片复制列表是全新的列表

使用切片可以看出对第二个列表修改不影响原始列表

xxxxxxxxxx
my_foods = ['pizza','falafel','carrot cake']
friend_foods = my_foods[:]
friend_foods.append('ice cream')
print(my_foods)
print(friend_foods)
xxxxxxxxxx
['pizza', 'falafel', 'carrot cake']
['pizza', 'falafel', 'carrot cake', 'ice cream']

直接引用=则操作的都是同一个列表

xxxxxxxxxx
my_foods = ['pizza','falafel','carrot cake']
friend_foods = my_foods
friend_foods.append('ice cream')
print(my_foods)
print(friend_foods)
xxxxxxxxxx
['pizza', 'falafel', 'carrot cake', 'ice cream']
['pizza', 'falafel', 'carrot cake', 'ice cream']
