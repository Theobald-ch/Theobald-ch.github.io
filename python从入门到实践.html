列表初级

定义

  • 列表是按顺序排列的元素组成。
  • 列表直接的元素可以没有任何关系。元素类型可以不同
  • 列表的元素也可以是个列表，即大列表里面有小列表。

示例

xxxxxxxxxx
bicycles = ['trek','cannondale','redline','specialized']
print(bicycles)

打印结果

xxxxxxxxxx
['trek', 'cannondale', 'redline', 'specialized']

列表元素操作

查询

  • 正向访问

列表是有序的，用元素位置访问即可。

注意列表里的元素位置是从0开始，0，1，2 ...

xxxxxxxxxx
bicycles = ['trek','cannondale','redline','specialized']
print(bicycles[0])
  • 倒序访问

-1代表最后一个元素，-2代表倒数第二个元素

xxxxxxxxxx
bicycles = ['trek','cannondale','redline','specialized']
print(bicycles[-1])

修改

将元素重新赋值

xxxxxxxxxx
motorcycles = ['honda','yamaha','suzuki']
print(motorcycles)
motorcycles[0] = 'ducati'
print(motorcycles)

结果：

xxxxxxxxxx
['honda', 'yamaha', 'suzuki']
['ducati', 'yamaha', 'suzuki']

增加

  • 在列表末尾添加元素

    print(motorcycles)

  • 根据元素位置插入元素

删除

  • 根据元素位置删除元素

    xxxxxxxxxx
    motorcycles = ['honda','yamaha','suzuki']

  • 使用pop()删除元素，并返回元素

    • pop()删除末尾元素

    print(motorcycles)
    print(popped_motorcycle)

    ['honda', 'yamaha']
    suzuki
    • pop(index) 删除index位置上的元素
    xxxxxxxxxx

    popped_motorcycle = motorcycles.pop(1)
    print(motorcycles)
    print(popped_motorcycle)

    ['honda', 'yamaha', 'suzuki']
    ['honda', 'suzuki']

  • 根据值删除元素(只会按顺序匹配第一个相同元素值删除)

列表解析

  • 列表解析一种python语法支持的列表操作方式
  • 优点是代码简洁，可将多行代码实现功能放在一行中
  • 面对初学者并非都会介绍列表解析，但是在阅读别人代码中很可能会碰到。

先通过for循环创建一个列表

xxxxxxxxxx
squares = []
for value in range(1,11):
   square = value**2
   squares.append(square)
print(squares)
xxxxxxxxxx
[1, 4, 9, 16, 25, 36, 49, 64, 81, 100]

列表解析写法

xxxxxxxxxx
squares = [value**2 for value in range(1,11)]
print(squares)

使用列表的一部分

  • 列表[f_index:l_index]：从f_index到l_index(不包括l_index)取出列表的子集。

继承

一个类继承另一个类时,它将自动获得另一个类的所有属性和方法。

原有的类称为父类，而新类称为子类。

一般把公共的部分抽成一个父类。

父类： 汽车 属性：年数

子类： 新能源车 属性：电池

汽油车 属性：油

xxxxxxxxxx
class Car():
   def __init__(self,year):
       self.year=year
   def get_car_message(self):
      print("year:"+self.year)
      
class ElectricCar(Car):
    def __init__(self,year,battery):
        super().__init__(year)
        self.battery=battery
    def get_electriccar_message(self):
        print("year:"+str(self.year)+",battery:"+str(self.battery))
class OilCar(Car):
    def __init__(self,year,oil):
        super().__init__(year)
        self.oil=oil
    def get_oilcar_message(self):
        print("year:"+str(self.year)+",oil:"+str(self.oil))
electric_car=ElectricCar(6,20)
electric_car.get_electriccar_message()
oil_car=OilCar(7,30)
oil_car.get_oilcar_message()
xxxxxxxxxx
year:6,battery:20
year:7,oil:30

WEB应用程序

介绍Django

  • Django是一个Web框架--一套用于帮助开发交互式网站的工具。
  • Django能够响应网页请求，还能让你更轻松地读写数据库等。

项目要求

  • 编写一个名为“学习笔记”的Web应用程序
  • “学习笔记”的主页对这个网站进行描述，并邀请用户注册或登录。
  • 用户登录后，就可创建新主题、添加新条目以及阅读既有的条目。
  • 让用户能够记录感兴趣的主题，并在学习每个主题的过程中添加日志条目。

建立虚拟环境

  • 每个项目都应该有一套独立的项目环境，主要目的是引用包隔离。

为项目新建一个目录，名为learning_log

注意：以下皆为windows下操作，linux命令可能操作方式不同。

激活虚拟环境

python -m venv ll_env
